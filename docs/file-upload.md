# 文件上传功能说明

## 功能描述
用户可以通过点击选择或拖拽方式上传Excel原价表文件，系统解析后返回结构化数据供后续编辑。

## 实现细节

### 前端实现
- 位置: `frontend/src/pages/UploadPage.tsx`
- 使用HTML5原生文件输入API
- 支持拖拽上传和点击选择两种方式
- 文件格式校验，只允许 `.xls` 和 `.xlsx` 格式
- 允许用户设置默认利润率，这个值会作为所有商品的初始值

### 后端实现
- 接口: `POST /api/parse`
- 使用 `xlrd` 读取Excel文件
- 自动检测表格结构、规格列、商品类型列
- 提取每个价格项，转换为结构化JSON返回前端
- 支持多Sheet、多表格自动检测

### 技术要点
1. **自动表格检测**: 自动寻找"规格"表头，支持规格列在不同位置
2. **数据去重分组**: 按Sheet分组，每个价格项单独记录保留完整信息
3. **错误处理**: 解析失败返回明确错误信息给前端显示
4. **CORS配置**: 支持前端跨域访问开发环境

## 使用说明
1. 打开首页，点击上传区域选择文件
2. 设置默认利润率（所有商品会先使用这个值）
3. 点击"开始解析"按钮
4. 解析成功后自动跳转到预览编辑页面

## 注意事项
- 文件大小没有限制，受浏览器和服务器配置约束
- 必须包含"规格"字样表头才能正确识别，如果没有会使用自动检测
