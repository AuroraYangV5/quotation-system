# 生成报价和下载功能说明

## 功能描述
用户预览和编辑完成后，点击生成按钮，后端根据自定义的每个商品利润率生成Excel报价表，然后浏览器下载保存到本地。

## 工作流程

1. **前端收集数据**
   - 收集所有商品（包括每个商品自定义的利润率）
   - 收集Sheet信息（标题、日期）保留原有格式
   - 发送JSON数据到后端 `/api/generate` 接口

2. **后端生成Excel**
   - 按Sheet分组整理数据
   - 计算每个商品的最终价格（原价 × (1 + 利润率/100)）
   - 创建新Excel工作簿，设置美观样式
   - 保留原Sheet结构和标题

3. **返回文件给浏览器下载**
   - 返回Blob二进制数据
   - 前端创建URL自动触发浏览器下载
   - 用户保存到本地即可使用

## 输出格式

- 文件格式: `.xlsx` (Excel 2007+)
- 文件名称: `报价表.xlsx`
- 样式：
  - 标题加粗放大，灰色背景
  - 表头灰色背景，居中对齐
  - 奇偶行不同背景色，方便阅读
  - 自动调整列宽适应内容
  - 完整边框

## 生成内容结构

```
Sheet: [原Sheet名称]
┌───────────────────────────────────┐
│  标题 报价表                       │ 合并单元格标题行
├───────────────────────────────────┤
│  利润率: +XX%             [日期]  │ 利润率信息行
├─────────┬─────────────────────────┤
│  规格   │  类型1  类型2  类型3...  │ 表头行
├─────────┼─────────────────────────┤
│  规格1  │   价格   价格   价格...  │ 数据行
│  规格2  │   价格   价格   价格...  │
└─────────┴─────────────────────────┘
```

## 实现细节

### 后端使用openpyxl生成
- 正确处理多Sheet多表格
- 保留原有标题和日期信息
- 应用统一美观样式模板
- 自动计算列宽

### 前端下载
- 使用 `Blob` 接收二进制数据
- 使用 `window.URL.createObjectURL` 创建下载链接
- 自动触发点击下载，不需要用户额外操作

## 使用说明
1. 确认所有商品利润率设置正确
2. 点击右上角"生成并下载"按钮
3. 等待处理完成（处理中按钮显示loading）
4. 浏览器自动下载 `报价表.xlsx`
5. 打开文件检查即可使用

## 注意事项
- 如果利润率有多个不同值，标题行会显示"自定义利润率 (多个不同值)"
- 所有自定义设置都会保留在生成的Excel中
- 生成是一次性的，可以多次生成不同版本下载
