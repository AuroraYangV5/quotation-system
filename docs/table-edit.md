# 表格预览编辑功能说明

## 功能概述

预览页面支持对解析后的表格数据进行编辑修改，包括规格、类型、原价、利润率、计算后价格等字段。

## 1. 字段编辑

### 支持编辑的字段

| 字段 | 说明 | 关联计算 |
|------|------|----------|
| 规格 (spec) | 商品规格名称 | 无 |
| 类型 (ruleName) | 商品类型 | 无 |
| 原价 (originalPrice) | 原始价格 | 修改后重新计算利润率 |
| 利润率 (%) | 加价百分比 | 修改后重新计算价格 |
| 计算后价格 | 最终售价 | 修改后反推利润率 |
| 自定义字段 | 动态字段 | 无 |

### 价格关联规则

- **修改利润率** → 自动重算 `计算后价格 = 原价 × (1 + 利润率/100)`
- **修改计算后价格** → 自动反推 `利润率 = ((计算后价格 - 原价) / 原价) × 100`
- **修改原价** → 保持当前利润率不变，重新计算价格
- 所有价格相关数值保留 **小数点后两位**

## 2. 删除行

- 点击行操作列的 **垃圾桶图标** 可删除该行
- 删除后该行数据不可恢复

## 3. 添加行

- 点击行操作列的 **+ 图标** 可以在该行下方插入新行
- 新行默认利润率使用全局利润率默认值 (20%)
- 新行 ID 为自动生成的时间戳字符串

## 4. 操作列说明

| 图标 | 功能 |
|------|------|
| + | 在当前行下方插入新行 |
| 🗑️ | 删除当前行 |
