# 在线预览编辑功能说明

## 功能描述
解析完成后在浏览器中直接预览所有商品和价格，可以在线编辑利润率，不需要下载到本地编辑。

## 界面布局

### 顶部操作栏
- 左侧显示统计信息：表格数量、商品规格总数
- 右侧有"返回上传"和"生成并下载"按钮

### 控制区（双卡片布局）
- **左侧**: 全局利润率滑块控制
- **右侧**: 批量设置（需要先勾选商品）

### Sheet切换
如果Excel有多个Sheet，会在表格上方显示Sheet标签栏，可以点击切换查看不同Sheet

### 搜索筛选
在表格上方提供搜索框：
- 支持按**规格名称**或**类型名称**搜索
- 输入关键词后表格实时筛选，只显示匹配结果
- 搜索框下方显示匹配结果数量统计
- 清空关键词恢复完整列表

### 数据表格
表格列：
| 列 | 功能 |
|----|------|
| 选择框 | 勾选用于批量修改或选择性导出 |
| 规格 | 商品规格名称 |
| 类型 | 商品类型/规则名称 |
| 原价 | 原始价格 |
| 利润率 | 可编辑输入框，当前设置值 |
| 计算后价格 | 计算结果，实时更新 |

### 表格操作
当前Sheet右上角提供快捷操作按钮：
- **全选/取消全选**：一键选中或取消选中当前Sheet所有商品

## 实现细节

### 前端技术
- **React 19 + TypeScript**: 类型安全
- **TanStack React Table**: 表格数据管理
- **Tailwind CSS**: 响应式样式
- **Shadcn UI 组件**: 基础组件
- **品牌色**: 使用 `#1a1a1a` 符合AGENT.md要求
- **微圆角**: 符合设计要求

### 状态管理
- 所有表格数据保存在React组件state中
- 使用sessionStorage持久化，刷新页面恢复
- 切Sheet不影响其他Sheet的数据修改

### 响应式设计
- 大屏: 双列布局显示控制卡片
- 移动端: 自动变为单列布局
- 表格支持横向滚动

## 使用提示
- 奇偶行使用不同背景色区分，方便阅读
- 多Sheet文件使用标签切换，一目了然
- 所有修改实时预览，价格立即更新

## 注意事项
- 数据保存在浏览器sessionStorage，关闭页面后会清除
- 大量数据（超过1000项）依然可以流畅编辑
